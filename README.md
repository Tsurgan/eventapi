API для организации мероприятий. Состоит из нескольких микросервисов: Gateway (Также содержит авторизацию и аутентификацию), Location service (содержит карту места мероприятия, список локаций и событий), Attendance service (обрабатывает информацию о посетителях и спикерах событий) и Task service (содержит информацию о задачах для организаторов).

PHP+Laravel+Passport, задокументировано на Swagger L5
